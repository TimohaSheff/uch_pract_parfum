1. СОЗДАНИЕ ПРОЕКТА И УСТАНОВКА БИБЛИОТЕК
─────────────────────────────────────────────────────────────────────────────

1.1. Инициализация Django проекта

Создан проект perfume_shop с использованием Django framework. 

Структура проекта:
• manage.py - точка входа для управления проектом
• requirements.txt - файл с зависимостями
• perfume_shop/ - основная директория проекта
  - settings.py - настройки проекта
  - urls.py - главный файл маршрутизации
  - wsgi.py, asgi.py - конфигурация для развертывания
• shop/ - приложение магазина
  - views.py - представления (контроллеры)
  - urls.py - маршруты приложения
  - templates/shop/ - шаблоны HTML
• static/shop/ - статические файлы (CSS, JavaScript)

1.2. Файл зависимостей (requirements.txt)

Django==4.2.7
Pillow==10.1.0

1.3. Основные настройки проекта

В файле perfume_shop/settings.py настроены:
• ROOT_URLCONF = 'perfume_shop.urls' - главный файл маршрутизации
• INSTALLED_APPS включает приложение 'shop'
• Настроены пути для статических файлов (STATIC_URL, STATICFILES_DIRS)
• Настроены пути для медиа файлов (MEDIA_URL, MEDIA_ROOT)
• Язык интерфейса: русский (LANGUAGE_CODE = 'ru-ru')
• Часовой пояс: Europe/Moscow

2.4. Представления (shop/views.py)

Основной код представлений:

def index(request):
    """Главная страница"""
    return render(request, 'shop/index.html')

def catalog(request):
    """Страница каталога"""
    return render(request, 'shop/catalog.html')

def contacts(request):
    """Страница контактов"""
    return render(request, 'shop/contacts.html')

2.5. Маршрутизация (shop/urls.py)

urlpatterns = [
    path('', views.index, name='index'),
    path('catalog/', views.catalog, name='catalog'),
    path('contacts/', views.contacts, name='contacts'),
]


3. СОЗДАНИЕ МЕНЮ НАВИГАЦИИ
─────────────────────────────────────────────────────────────────────────────

3.1. Базовый шаблон (shop/templates/shop/base.html)

Реализованные элементы навигации:

1. ЛОГОТИП И НАЗВАНИЕ САЙТА
   • Иконка парфюма (Font Awesome: fa-spray-can)
   • Название: "ПарфюмЭлит"
   • Оба элемента являются ссылкой на главную страницу
   • При нажатии осуществляется переход на главную ({% url 'index' %})

2. МЕНЮ НАВИГАЦИИ
   • Ссылка "Главная" → переход на главную страницу (/)
   • Ссылка "Каталог" → переход на страницу каталога (/catalog/)
   • Ссылка "Контакты" → переход на страницу контактов (/contacts/)

3. КНОПКИ АВТОРИЗАЦИИ
   Для неавторизованных пользователей:
   • Кнопка "Вход" → переход на страницу входа (/admin/login/)
   • Кнопка "Регистрация" → переход в админ-панель (/admin/)
   
   Для авторизованных пользователей:
   • Отображение имени пользователя
   • Кнопка "Выход" → выход из системы (/admin/logout/)

Структура навигации в HTML:

<nav class="navbar">
    <a href="{% url 'index' %}" class="logo-section">
        <div class="logo"><i class="fas fa-spray-can"></i></div>
        <span class="site-name">ПарфюмЭлит</span>
    </a>
    <ul class="nav-menu">
        <li><a href="{% url 'index' %}">Главная</a></li>
        <li><a href="{% url 'catalog' %}">Каталог</a></li>
        <li><a href="{% url 'contacts' %}">Контакты</a></li>
    </ul>
    <div class="auth-buttons">
        {% if user.is_authenticated %}
            <a href="#" class="btn">{{ user.username }}</a>
            <a href="{% url 'admin:logout' %}" class="btn">Выход</a>
        {% else %}
            <a href="{% url 'admin:login' %}" class="btn">Вход</a>
            <a href="{% url 'admin:index' %}" class="btn">Регистрация</a>
        {% endif %}
    </div>
</nav>
