# Инструкции по применению миграций

После создания модели CustomUser необходимо выполнить следующие команды:

1. Создать миграции:
```bash
python manage.py makemigrations
```

2. Применить миграции:
```bash
python manage.py migrate
```

3. Создать суперпользователя (если нужно):
```bash
python manage.py createsuperuser
```

**ВАЖНО:** После изменения AUTH_USER_MODEL необходимо:
- Удалить старую базу данных (db.sqlite3) если она уже существует
- Или создать новую базу данных с нуля
